# Comments Page

## Basic task - Create add comment block
1. block contains form for adding comment:
      - name, text;
      - the ability to specify a date or leave it blank;
      - submit button;
2. list of comments;
3. delete comment button;
4. like button.

## Used technology
- HTML;
- JavaScript;
- SASS;
- Webpack;
